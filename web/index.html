Bot トークン設定

Discord Developer Portal で取得した Bot トークンを入力してください。
トークンはこの PC 内の config.json にのみ保存され、外部には送信されません。

トークンの取得方法
  1. Discord Developer Portal で New Application → Bot
  2. Bot ページの「Reset Token」でトークンを表示してコピー
  3. 同じページの Privileged Gateway Intents で SERVER MEMBERS INTENT を ON
  4. OAuth2 → URL Generator で bot スコープ + 権限 Move Members View Channels Send Messages Connect を選び、生成された URL からサーバーに招待
