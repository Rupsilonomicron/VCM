# 変更履歴

## v1.6.0 (2026-07-08)

- **ドラッグ範囲選択**: パネルの余白からドラッグすると選択枠が表示され、
  枠に触れたメンバーを一括選択できる。そのままドラッグ＆ドロップで全員をまとめて移動。
  - Ctrl（または Shift）を押しながら囲むと、既存の選択に追加
  - 余白をクリック（ドラッグせず）すると選択をすべて解除

## v1.5.2 (2026-07-08)

- **設定画面の「VOICEVOX の場所」の「参照…」ボタンを廃止**（使いにくかったため）。
  代わりに、入力欄の下にインストール先の記入例を表示するようにした。

## v1.5.0 (2026-07-08)

- **設定画面の追加**: ヘッダーの「⚙ 設定」から、`config.json` を直接編集しなくても
  次の項目を変更できるようにした。
  - 起動時に選ぶサーバー（参加中のサーバーから選択）
  - ポート番号（変更は次回起動時に反映）
  - VOICEVOX の場所（変更するとその場で再検出。再起動不要）
  - Bot トークンの設定・変更（従来のトークン画面をここから開ける）
  - ※ 安全のため `host` は引き続き GUI からは変更できません
    （変更するとネットワークに公開されてしまうため）。

## v1.4.2 (2026-07-08)

セキュリティ強化と安定性修正のアップデート。

- **セキュリティ: ブラウザ経由のローカルAPI攻撃を遮断**:
  VCM起動中に悪意あるWebページを開いた場合に、そのページからVCMのAPIを実行されたり
  （CSRF）、WebSocket経由でサーバー名・メンバー名などの情報を読み取られたりする
  可能性があった。Origin/Hostヘッダーの検証を追加し、localhost以外からの
  アクセスを拒否するようにした。
- **セキュリティ: `config.json` の `host` を変更しないよう注意書きを追加**
  （変更すると認証なしの全機能がネットワークに公開されるため）。
- GUIを全て閉じてもまれにアプリが終了しなくなる競合を修正。
- メインVC変更時、変更前のVCでの読み上げ中の1件も即座に打ち切るように修正。
- 設定ファイル（config.json / presets.json / tts.json）の保存をアトミックにし、
  書き込み中のクラッシュで設定が消えないようにした。
- アップデート後の自動再読み込みに3分の上限を設け、失敗時は案内を表示するようにした。

## v1.4.1 (2026-07-07)

- **散開先の決め方を「D&Dの記録」方式に変更**（v1.4.0の多数決方式を置き換え）:
  チーム見出しをVCへドラッグして移動したとき、そのVCがチームの「散開先」として記録される。
  散開はチームの全メンバーを記録済みの散開先VCへ移動する（未記録のチームはスキップ）。
  - VC一覧に「⛺ チーム名」バッジを表示し、どのVCがどのチームの散開先か一目で分かる
  - 散開先はチーム編成プリセットにも保存・復元される
  - 散開先が1つも記録されていない状態で散開すると、記録方法を案内するメッセージを表示

## v1.4.0 (2026-07-07)

- **集合・散開をチーム単位に変更**: 集合時に「各チームがどのVCにいたか」を記録するようになった
  （従来は個人ごとの位置を記録）。散開はチームの全メンバーを記録したチームVCへ移動する。
  - チームのVCは「メインVC以外で最も多くのメンバーがいたVC」で判定
  - 集合前に個人が別のVCへはぐれていても、散開でチームのVCへ合流する
  - 集合後にチームへ加入したメンバーも、散開でチームのVCへ一緒に移動する
  - チーム未所属のユーザーを動かさない点は従来どおり

## v1.3.3 (2026-07-07)

- **アップデート後、アプリ終了時にcmdウィンドウが閉じない問題を修正**:
  更新用スクリプトが `start` でバッチを直接起動していたため、cmd が対話モード
  （`/K` 相当・終了後もプロンプト表示のまま）で開いていた。`cmd /c` を明示して、
  アプリ終了と同時にウィンドウも閉じるようにした。
  ※この修正は v1.3.3 以降からのアップデートで有効（v1.3.2 から v1.3.3 への
  アップデート時のみ、最後にもう一度ウィンドウが残るので手動で閉じてください）。

## v1.3.2 (2026-07-07)

- **アップデート後にタブとウィンドウが増える問題を修正**:
  アップデート後の再起動で新しいタブを開かず、アップデートを実行したタブがそのまま
  自動で新しいバージョンの画面に切り替わるようにした。
  これまでは古いタブが新アプリに再接続し続けるため、新しいタブを閉じても
  アプリ（cmdウィンドウ）が終了しないことがあった。

## v1.3.1 (2026-07-07)

- **初回起動時にブラウザが「接続拒否」画面になる問題を修正**:
  ブラウザを開くタイミングを start_bot.bat からアプリ本体（サーバー待ち受け開始後）に変更。
  起動に時間がかかる初回でも、画面が確実に表示されるようになった。
  あわせて `config.json` でポートを変更している場合も正しい URL が開くように。

## v1.3.0 (2026-07-07)

- **ワンクリックアップデート**: 更新通知をクリックすると確認画面が開き、
  「アップデートして再起動」を押すだけで新バージョンのダウンロード・適用・再起動まで自動で行われる。
  設定（トークン・プリセット・辞書など）はそのまま引き継がれる。
  自動アップデートは配布版のみ（ソースから実行している場合は従来どおりリリースページへのリンク）。

## v1.2.0 (2026-07-07)

- **シャッフルのピン留め**: チーム内のメンバーカードにマウスを乗せると📌ボタンが表示され、
  クリックでピン留めできる。ピン留めしたメンバーはシャッフルしても現在のチームから動かない
  （チームリーダーの固定などに）。人数はピン留め分も含めて均等に振り分けられる。
  全員ピン留めされている場合はシャッフルできない。
- **更新通知**: 起動時に GitHub の最新リリースを確認し、新しいバージョンが公開されていると
  GUI ヘッダーに「⬆ vX.X.X が公開されています」リンクを表示（クリックでリリースページを開く）。
  オフラインでも動作に影響なし。参照先は config.json の `github_repo` で変更可能。

## v1.1.1 (2026-07-05)

- **/voice のキャラクター選択を見やすく改善**: 選択メッセージに、どのページにどのキャラクターがいるか一覧できる表（ページごとの列表示・1行1キャラ）を追加。
  どのページに目当ての声があるか、ページをめくらなくても分かるようになった。
  - 一覧はキャラクターが1ページに収まるときは表示されない。
  - スタイル選択に進むと一覧は消え、「◀ キャラクター選択に戻る」で戻ると再表示される。

## v1.1.0 (2026-07-04)

- **参加希望の受付機能**: コントロールバーに「✋ 参加希望を募る」ボタンを追加（メインVCとチームが1つ以上あるとき押せる）。
  - 押すと bot がメインVCの内蔵チャットに「✋ 参加希望」ボタン付きの募集メッセージを設置。
  - メインVCに接続しているユーザーだけがボタンを使用でき、押すと本人にのみ見えるチーム選択メッセージが表示される。
  - チームを選ぶとGUIのチームに所属し（1人1チーム・選び直し可）、選択メッセージは自動的に消える。
    60秒間選択しなかった場合も自動的に消える。
  - 募集中はボタンが「✋ 募集を締め切る」に変わり、押すと募集メッセージを削除。
    メインVCの変更・解除時や bot 終了時も募集メッセージは自動で撤去される。
  - bot に招待時の `Send Messages` 権限が必要（トークン設定画面の案内にも追記）。

## v1.0.1 (2026-07-04)

- **チームシャッフル機能**: チームパネルに「🔀 シャッフル」ボタンを追加（「＋チーム追加」の右隣）。
  チームに所属している全ユーザーを、既存チームへランダムかつ均等（人数差は最大1人）に振り分け直す。
  押すたびに再シャッフルされる。チームが2つ以上かつ所属メンバーがいるときだけ押せる。
- **読み上げ辞書の誤爆修正**: 英数字のみの単語（例:「w」「VCM」）は、前後が英数字でない場合だけ置換するように変更。
  「w」を登録しても「Windows」「web」など英単語の中の w には反応しなくなった。
  日本語の直後の w（「きたw」など）はこれまで通り読み替えられる。
- 1文字の登録語は連続をまとめて1回だけ読むように変更（「wwww」→「わら」×1）。
- 日本語の辞書語（「鯖」→「サーバー」など）の動作は従来通り。
- チーム追加ボタンをチームパネルの横幅いっぱいの表示に変更。

## v1.0.0 (2026-07-03)

初版リリース。

**VC編成**
- サーバー内のVC一覧・接続ユーザー（名前＆アイコン）・チームのリアルタイム表示
- VC一覧はカテゴリー別のグループ表示（Discordクライアントと同じ並び順）
- メンバーカードのドラッグ＆ドロップ（VC移動／チーム割当・1人1チーム）
- チーム作成／リネーム／削除、チームヘッダをVCへドラッグで一括移動
- メインVC設定＋集合／散開（集合直前の位置へ復元）
- チーム編成プリセットの保存／呼び出し／削除
- 複数サーバー対応（ヘッダのプルダウンで切替、状態はサーバーごとに独立）

**読み上げ（VOICEVOX連携）**
- VOICEVOX検出時のみ有効（無くてもVC編成機能はすべて使える）
- メインVCへ入室し、入室中VCの内蔵チャットを読み上げ
- /voice コマンドでユーザーごとの声（キャラ→スタイル）設定
- スキップ／キュー消去／テスト再生、読み上げ辞書（サーバーごとに保存）

**その他**
- トークン設定GUI（初回起動時に案内、マスク表示・検証付き）
- 実行環境同梱の配布形式（Pythonインストール不要）
- GUIブラウザを全て閉じるとアプリ自動終了
- 正式名称を VoiceChatMover に統一
